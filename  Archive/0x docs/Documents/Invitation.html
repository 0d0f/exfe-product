+ (Object) current

Current invitation represent the credential of visit Exfe with an invitation token.

When a user visit Exfe without login, but with an invitation token, we need to use the corresponding Invitation object to identify the user. Get abilities for the user via Ability.new(Invitation.current).

def serializable_hash(options = nil)
  options ||= {}
  options = options.merge(:include => :invited_identity)

  unless invited_user == User.current
    options[:except] ||= []
    options[:except] += PRIVATE_ATTRIBUTES
  end

  super
end
