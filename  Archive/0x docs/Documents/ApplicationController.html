- (Object) after_sign_in (private)

Clear invitation token from session to ensure current_ability be created for the user who just signed in, in the next request.

We’re not going to clear Invitation.current since it may be used in current request later, to check if it’s a “sign up via invitation” case.

def current_identity

  if @event
    current_user.identity_for(@event)
  else
    current_user.identity
  end
end

- (Object) load_current_invitation (private)

Load current invitation if there’s invitation token stored in session, or clear it if invitation token no longer available.

Current invitation will be used when:

  • Visit event page.

  • Signup as an invited user.

def load_current_invitation
  Invitation.current = if session[:invitation_token]
                         Invitation.find_by_token(session[:invitation_token])
                       else
                         nil
                       end
end

def render_error(exception)
  logger.fatal exception
  logger.fatal exception.backtrace.join("\n")

  respond_to do |format|
    format.html do
      render 'pages/error', :status => :internal_server_error
    end

    format.json do
      render :json => {:error => 'Internal server error.'}
    end
  end
end

- (Object) set_gettext_locale (private)

Set current locale to :zh_CN in controller thread.

This is required since the fast_gettext initializer runs in different thread.

If need set locale based on HTTP_ACCEPT_LANGUAGE, just remove this method so the one defined in gettext_i18n_rails will be used.
